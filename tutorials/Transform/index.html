CSS Transform

Translate (x, y)

translate

translate - сдвигает элемент на новое место относительно начального положения объекта вправо и вниз, используя координаты X и Y.

Scale (x, y)

scale

scale - масштабирует элементы, делая их больше или меньше. По умолчанию scale(1, 1). При значениях больше 1 объект увеличивается, от 0 да 1 - уменьшается, при отрицательных значениях отображается зеркально

Rotate (n+deg)

rotate

rotate - поворачивает элементы на заданное колличество градусов, по умолчанию rotate(0deg)

Skew (x+deg, y+deg)

skew

skew - деформирует (искажает) стороны объекта относительно координатных осей. По умолчанию skew(0deg, 0deg)

Matrix (a, c, b, d, x, y)

matrix

matrix - позволяет объединить несколько функций 2D-трансформаций в одной. По умолчанию matrix(1, 0, 0, 1, 0, 0)
Значение a изменяет масштаб по горизонтали. Значение от 0 до 1 уменьшает элемент, больше 1 — увеличивает.
Значение c деформирует (сдвигает) стороны элемента по оси Y, положительное значение — вверх, отрицательное — вниз.
Значение b деформирует (сдвигает) стороны элемента по оси X, положительное значение — влево, отрицательное — вправо.
Значение d изменяет масштаб по вертикали. Значение меньше 1 уменьшает элемент, больше 1 — увеличивает.
Значение x смещает элемент по оси X, положительное — вправо, отрицательное — влево.
Значение y смещает элемент по оси Y, положительное значение — вниз, отрицательное — вверх.

Множественные 2D трансформации

multi

Множественнaя 2D трансформация - позволяет объединить несколько функций 2D-трансформаций в одной. Пр. transform: translate(30px, 30px) rotate(45deg) scale(0.8);

Transform-origin

transform-origin

transform-origin - позволяет сместить центр трансформации, относительно которого происходит изменение положения/размера/формы элемента. По умолчанию center (50% 50%)

Perspective, perspective-origin

perspective-origin

perspective-origin - установка глубины и смена точки начала координат. По умолчанию perspective: 0

Translate3d (x, y, z)

translate3d

Translate3d - по аналогии со своим 2D братом, это значение задает перемещение элемента, но в 3D-пространстве. Помимо оси X и Y добавляется ось Z. По умолчанию translate3d(0px, 0px, 0px)

Scale3d (x, y, z)

scale3d

Scale3d - задает трехмерное масштабирование по одной из осей X Y Z. Фактически повторяет функционал своего 2d аналога. По умолчанию scale3d(1, 1, 1)

Rotate3d (x, y, z, deg)

rotate3d

Rotate3d - вращает элемент относительно трех осей. Элемент поворачивается под углом, задаваемым последним параметром относительно вектора направления x,y,z. По умолчанию rotate3d(1, 1, 1, 0deg)

Matrix3d (n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n)

matrix3d

Matrix3d - задает трехмерное преобразование как однородную матрицу размером 4×4 с шестнадцатью значениями в столбцах. Генератор 3D матриц - http://ds-overdesign.com/transform/matrix3d.html

Множественные 3D трансформации

multi3d

Множественные 3D трансформации - позволяет объединить несколько функций 3D-трансформаций в одной. Пр. transform: translate3d(30px, 30px,30px) rotate3d(1,1,1,30deg);

Transform-style

transform-style

Transform-style - определяет, как дочерние элементы должны отображаться в 3D-пространстве. По умолчанию transform-style:flat

Backface-visibility

front

back

Backface-visibility - отображение обратной стороны объекта. По умолчанию backface-visibility: visible;
